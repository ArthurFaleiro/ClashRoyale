Tecnologia: Power BI, DAX (Data Analysis Expressions)

Domínio: Business Intelligence / Game Design

Análise estatística de performance e balanceamento de cartas baseada nos dados do jogo da Supercell.

📈 Componentes do Relatório
Modelagem de Dados: Relacionamento entre tabelas de "Cartas", "Troféus" e "Win Rate".

KPIs (Indicadores):

Uso de elixir médio por deck.

Taxa de vitória por arena.

Sinergia entre cartas específicas.

Visualizações: Gráficos de barras dinâmicos, treemaps de popularidade e filtros por raridade (Comum, Rara, Épica, Lendária).
